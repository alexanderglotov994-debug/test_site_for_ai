# Blind AI Battle (GitHub Pages + Google Sheets)

Статический сайт для слепого A/B-теста ответов ИИ-агентов:
- Пользователь проходит 7 случайных вопросов.
- На каждом вопросе видит 2 анонимных ответа и выбирает лучший.
- Порядок вопросов и расположение ответов случайные.
- Отдельная админ-страница для:
  - редактирования контента вручную,
  - массовой загрузки базы из таблицы (CSV/TSV),
  - просмотра статистики по датам,
  - отчётов по агентам, системным промптам и пользовательским промптам.

## 1) Структура проекта

- `index.html` — публичный тест
- `admin.html` — админка по отдельной ссылке
- `assets/styles.css` — стили
- `assets/app.js` — логика публичного теста
- `assets/admin.js` — логика админки
- `config.example.js` — пример конфига
- `config.js` — ваш локальный конфиг (нужно создать из примера)
- `apps-script/Code.gs` — backend API для Google Apps Script
- `templates/dataset-template.csv` — шаблон для массового импорта

## 2) Настройка Google Apps Script API

1. Создайте Google Таблицу.
2. Откройте `Расширения -> Apps Script`.
3. Вставьте код из `apps-script/Code.gs`.
4. В `Code.gs` укажите ID таблицы в `SPREADSHEET_ID`.
5. Деплой:
   - `Deploy -> New deployment -> Web app`
   - Execute as: `Me`
   - Who has access: `Anyone`
6. Скопируйте URL вида `https://script.google.com/macros/s/.../exec`.

Скрипт сам создаст листы при первом запросе:
- `items`
- `sessions`
- `votes`

## 3) Настройка фронта

1. Скопируйте `config.example.js` в `config.js`.
2. В `config.js` вставьте `API_BASE_URL` вашего Apps Script.
3. Убедитесь, что `QUESTIONS_PER_SESSION = 7`.

## 4) Публикация на GitHub Pages

1. Залейте проект в репозиторий.
2. В `Settings -> Pages` выберите ветку (`main`) и папку (`/root`).
3. Сайт будет доступен по URL GitHub Pages.

Публичная страница:
- `https://<user>.github.io/<repo>/`

Админка:
- `https://<user>.github.io/<repo>/admin.html`

## 5) Загрузка базы таблицей

Поддерживаемые колонки в таблице:
- `Модель`
- `Промт`
- `Вопрос`
- `Ответ модели`

Как импортировать:
1. Откройте админку (`/admin.html`).
2. Скачайте шаблон `Скачать шаблон CSV` или используйте [templates/dataset-template.csv](templates/dataset-template.csv).
3. Загрузите CSV-файл или вставьте таблицу из Google Sheets в поле импорта.
4. Нажмите `Импортировать таблицу`.

Логика импорта:
- Группировка идёт по полю `Вопрос`.
- Если для вопроса есть минимум 2 строки (2 модели), автоматически создаются пары A/B.
- При 3+ моделях на один вопрос создаются все попарные комбинации.
- Опция `Перед импортом деактивировать текущие активные кейсы` заменяет активную базу новой.

## 6) Важно

- Админка без авторизации (как вы просили), только по отдельной ссылке.
- Для скрытия URL админки можно переименовать `admin.html` в любой уникальный путь.
- Данные доступны всем, у кого есть прямой API URL и доступ к странице.
